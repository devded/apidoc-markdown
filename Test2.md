# API Documentation

## API 01: Create User

- **Endpoint:** `https://makewish.com/api/add`
- **Request:**
  ```json
  {
    "name": "D",
    "age": 17
  }
  ```
- **Response:**
  ```json
  {
    "name": "D",
    "age": 17
  }
  ```

---
